# 微动计划 - 公众号自动化运营项目

每天微动5分钟，身体自己会回报。

## 项目概述

这是一个微信公众号自动化运营项目，面向所有久坐、缺乏运动的人群（18-40岁），提供碎片化微运动方案、身体问题自救指南、运动装备推荐和健康热点解读。

## 账号信息

| 项目 | 内容 |
|------|------|
| 公众号名称 | 微动计划 |
| 定位 | 每天微动5分钟，身体自己会回报 |
| 目标读者 | 18-40岁久坐/缺乏运动人群 |
| 发布频率 | 日更（每天1篇，每周7篇） |
| GitHub 仓库 | https://github.com/nie3360/weidongjihua |

## 栏目设计

| 栏目 | 频率 | 方向 |
|------|------|------|
| 「微动5分钟」 | 每周3篇 | 全场景碎片化微运动方案 |
| 「身体求救信号」 | 每周2篇 | 久坐生活方式导致的身体问题与运动处方 |
| 「装备避坑指南」 | 每周1篇 | 运动/恢复装备测评与推荐 |
| 「热点×微动」 | 每周1篇 | 健康/运动/生活方式热点解读 |

## 项目结构

```
├── config/                  # 账号配置
│   └── account_config.json  # 账号定位、栏目、变现路径
├── scripts/                 # 自动化脚本
│   ├── fetch_hot_topics.py  # 微博热搜+知乎热榜抓取
│   └── filter_topics.py     # 运动/健康/生活话题筛选
├── templates/               # 栏目写作模板
│   ├── micro_movement.md    # 「微动5分钟」模板
│   ├── body_signal.md       # 「身体求救信号」模板
│   ├── gear_guide.md        # 「装备避坑指南」模板
│   └── hot_micro.md         # 「热点×微动」模板
├── articles/                # 文章归档（按日期）
│   └── auto-reply/          # 自动回复文章
├── assets/images/           # 配图素材
└── .gitignore
```

## 日常使用

### 1. 抓取今日热搜

```bash
pip install requests
python scripts/fetch_hot_topics.py
```

### 2. 筛选运动/健康相关话题

```bash
python scripts/filter_topics.py
```

### 3. 撰稿 & 发布

- 根据筛选结果选择话题，或按栏目模板自行选题
- 使用对应栏目模板撰写文章
- 生成 HTML 排版后粘贴到公众号后台发布
